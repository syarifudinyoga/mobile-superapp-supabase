# 🛠 Prerequisites

Install

- Node.js LTS
- npm
- Xcode (macOS)
- Watchman
- Expo CLI (via `npx`)
- Git

Check version

```bash
node -v
npm -v
npx expo --version
```

---

# 🚀 Getting Started

Create project

```bash
npx create-expo-app@latest mobile-superapp
```

Go into project

```bash
cd mobile-superapp
```

Run

```bash
npm start
```

Android

```bash
npm run android
```

iOS

```bash
npm run ios
```

Web

```bash
npm run web
```
